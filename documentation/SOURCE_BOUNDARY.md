# Source Boundary

Included:

- Original `.xls` analysis workbooks from the paper folder.
- Modernized `.xlsx` conversions of those workbooks.
- Original appendix `.doc` files that document methods and calculations.
- Text extracts from the appendix files.
- A reference copy of the published paper for local package review.

Excluded:

- Draft manuscripts and older article variants.
- Reviewer responses, transmittal forms, correspondence, and email/text messages.
- Presentations and slide decks.
- Bibliography files, background/reference PDFs, policy-paper side folders, and unrelated paper material.
- Raw MnDOT detector feeds, because no raw feed dataset was found in this paper folder and this package is scoped to the derived analysis files that were present.
